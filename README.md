# Jobflix (Level-up) Platform

Welcome to the Jobflix repository! This is a modern, feature-rich web application built to beautifully showcase the 7 core products and features of the Jobflix platform.

## Features

- **Modern Landing Page**: A beautifully crafted hero section with dynamic floating nodes and premium aesthetics.
- **Interactive World Map**: A visually stunning component that highlights our global footprint and user interactiveness.
- **Bento Grid Layout**: Content-rich grid showcasing our 7 flagship products and features with engaging hover states and micro-animations.
- **Theme-Aware Design**: Seamless Light and Dark mode transitions paired with a brand-focused color palette.
- **Typography and Aesthetics**: Utilizing Sora for bold headings and DM Sans for highly readable body copy, paired with glassmorphism and subtle gradients.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: Tailwind CSS, Vanilla CSS, Framer Motion, and Aceternity UI components.
- **Database / ORM**: Prisma (connected via Neon/AWS).

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Open the app**:
   Navigate to [http://localhost:3000](http://localhost:3000) with your browser to see the result.
